# Calibre-Web 扩展版（课程设计项目）

本项目基于开源项目 **Calibre-Web** 进行二次开发，作为《软件工程 / 课程设计》实践项目，
在原有电子书管理与阅读功能的基础上，新增了多项扩展功能，并对系统进行了模块化设计与测试。

---

## 一、项目背景

Calibre-Web 是一个基于 Python Flask 的电子书管理与阅读系统，
支持在线阅读、书籍管理、用户管理等功能。

本项目在其基础上进行了功能扩展，重点围绕 **数据采集、智能分析、系统集成测试** 展开。

---

## 二、新增功能说明（课程设计）

### 1️⃣ 图书爬虫模块
- 支持从指定网站爬取图书信息
- 自动下载书籍文件（如 EPUB / PDF）
- 支持批量导入到 Calibre 本地书库
- 提供接口供 Web 前端触发导入

### 2️⃣ 阅读进度同步
- 记录用户的阅读位置与进度百分比
- 支持多设备之间的阅读进度同步
- 基于数据库持久化保存阅读状态

### 3️⃣ AI 自动摘要模块
- 对书籍内容进行智能分析
- 自动生成书籍摘要信息
- 为书籍详情页和推荐系统提供语义支持

### 4️⃣ 书籍推荐系统
- 基于用户阅读行为进行推荐
- 支持内容推荐与协同过滤推荐
- 提供独立测试模块验证推荐效果
